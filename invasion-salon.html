Full-Stack MERN App: Invasion Salon

A hair salon website and scheduling app I built using Mongoose, Express, React, and Node.

With only five weeks of coding coursework, I designed a fully functional website using Mongoose, Express, React, and Node.

This website has full Create, Read, Update and Delete operations. I designed it as a blueprint website for a real salon owned by a friend of mine. The most rewarding part of the project was being able to incorporate Maybe's unique personality into the styling of the website.

Although the current version allows users to create, read, update and delete appointments, the appointment information is not password protected. Future updates will include user authentication complete with usernames and passwords, and will send the owner a text message alert when someone books a new appointment.
